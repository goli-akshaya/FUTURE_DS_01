# FUTURE_DS_01 — Business Sales Performance Analytics

![Dashboard Preview](futureinterns%20task%201.jpeg)

**Internship:** Data Science & Analytics, Future Interns  
**Task:** Task 1 – Business Sales Performance Analytics  
**Tool used:** Microsoft Excel (Power Query, PivotTables, DAX)

## 📊 Overview
Interactive dashboard analyzing the US Superstore dataset (9,994 orders, 2014–2017) to uncover revenue trends, regional performance, category profitability, and discount impact on profit.

## 🔢 Key Metrics
- **Total Sales:** $2.30M (↑51.4% over 4 years)
- **Total Profit:** $286K (↑88.6% growth)
- **Total Orders:** 9,994
- **Total Customers:** 793 (avg $459 per order)
- **Profit Margin:** 12%

## 🔍 Key Insights
- West region leads overall sales performance; California generates the highest order volume and total profit
- Technology is the highest-performing category by profit
- Discounts above 30% significantly reduce profitability — returns turn negative beyond that threshold
- Standard Class shipping handles the highest order volume; Same Day delivery averages under 1 day
- Canon Advanced Copier is the top profit-generating product

## 🛠️ Process
- Cleaned and transformed raw order data using Power Query (fixed a date-extraction bug affecting Dec 2017 records)
- Built KPI cards using linked shapes/text boxes with custom number formatting
- Created PivotTables and PivotCharts for monthly trends, regional sales, and category/sub-category breakdowns
- Applied conditional formatting to highlight discount-profit relationships
- Added interactive slicers for Region and Year filtering
